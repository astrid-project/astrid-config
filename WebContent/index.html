Verifoo RESTful Web Service

This web service has the following resources
  • deployment
  • converter
  • simulation
The deployment resource recives an XML file containing a NFV element that describes some service chains and a physical netwok. It returns the correspondent deployment. The converter resource recives also an XML file containing a NFV element with a ParsingString element which is the model returned by z3. The simulation resource allows the interaction with SONATA.
